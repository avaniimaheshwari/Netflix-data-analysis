# Netflix/Movie Data Analysis

##  Project Overview
This project analyzes a dataset of movies to uncover trends in popularity, genres, and voting patterns. The goal was to practice data cleaning, visualization, and exploratory data analysis (EDA) using Python.

##  Dataset
The analysis is based on the `mymoviedb.csv` file, which includes data such as:
- **Movie Title & Overview**
- **Release Date**
- **Popularity & Vote Counts**
- **Genres** (Action, Drama, Sci-Fi, etc.)
- **Original Language**

## Technologies Used
- **Python**
- **Pandas** (for data manipulation)
- **NumPy** (for numerical operations)
- **Matplotlib & Seaborn** (for data visualization)
- **Jupyter Notebook**

##  Key Findings

- Most Popular Genre: Thriller 
- Release Trends:2020
